# Tempus Regisztrációs Alkalmazás

Ez a projekt egy modern, reszponzív, kettéosztott képernyővel (Split Screen) rendelkező regisztrációs felület, amely React és Bootstrap használatával készült.

## 🛠 Technikai specifikációk

A fejlesztés során az alábbi környezetet és technológiákat alkalmaztuk:

* **React:** `^19.2.6`
* **Node.js:** `v20.19.2`
* **Stílusozás:** Bootstrap 5
* **Build eszköz:** Vite

## 📚 Források és külső eszközök

## 🤖 AI Asszisztencia

A fejlesztési folyamat során a **Gemini** és **DuckDuckGo** szolgáltatásokat használtam a technikai problémák megoldására és a kód struktúrájának optimalizálására. A részletes beszélgetéseket és az AI-jal közösen folytatott chat-et `ai_gemini.txt` fájl tartalmazza, terjedelmi okokból innen kivettem.
